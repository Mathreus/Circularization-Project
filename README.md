# Projeto de Circularização Contínua

# Bem vindos!

Este repositório é dedicado a automações nas atividades da Circularização. O objetivo principal é transformar a forma como as análises são realizadas, substituindo tarefas manuais e repetitivas por análises rápidas, consistentes e escaláveis, permitindo que a equipe dedique mais tempo a confirmação com os clientes, ganhando assim um salto de produtividade! 

Aqui, você encontrará os códigos e scripts desenvolvidos para realizar a análise de riscos dos distribuidores, construindo assim uma equipe de Circularização mais produtiva, ágil e precisa!

# Estrutura e Organização do Repositório:

# 1) Segregação por itens:

Os testes estão organizados por itens do relatório analítico, com o objetivo de agilizar a localização do script adequado para cada item. Dessa forma, estão segregados por:

- Código Base: Gera os relatórios base como: Faturamento, Devolução e Bonificação
- Análise: São os códigos que irão retornar os resultados para preenchimento dos itens da aba "Análise"
- Seleção: São os códigos que irão retornas os resultados para auxiliar na seleção dos clientes a serem circularizados

# 2) Para executar um teste:
- Navegue até a pasta de interesse;
- Localize o script com o código do teste desejado;
- Copie o código do teste e cole no ambiente de programação utilizado para executar o teste
- Leia e siga as instruções escritas no início do código, lá existem dicas de como executar de maneira correta
- 
OBS: É importante destacar que os códigos podem necessitar de atualizações e refinamentos contínuos para se manterem alinhados com eventuais mudanças nos processos auditados e assegurar a precisão e confiabilidade dos resultados!

# 3) Resultados esperados: 
A implementação bem sucedida desta automação busca:

- Reduzir o tempo gasto em testes manuais, demorados e repetitivos;
- Aumentar a cobertura de checagem em processos críticos;
- Identificar anomalias com maior precisão e antecedência
