# Projeto de Circularização Contínua

# Bem vindos!

Este repositório é dedicado a automações nas atividades da Circularização. O objetivo principal é transformar a forma como as análises são realizadas, substituindo tarefas manuais e repetitivas por análises rápidas, consistentes e escaláveis, permitindo que a equipe dedique mais tempo a confirmação com os clientes, ganhando assim um salto de produtividade! 

Aqui, você encontrará os códigos e scripts desenvolvidos para realizar a análise de riscos dos distribuidores, construindo assim uma equipe de Circularização mais produtiva, ágil e precisa!
